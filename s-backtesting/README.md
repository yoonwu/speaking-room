# s-backtesting — 다이버전스 복합 변곡 전략 백테스터

RSI 다이버전스를 **조건**으로, 자리 필터(수평 S/R · 피보나치 · 볼린저)를 **위치 검증**으로, 캔들 반전 패턴 / MACD를 **방아쇠**로 조합한 변곡 포착 전략의 전수(全數) 그리드서치.

```
[조건]   RSI 다이버전스 발생 (regular, 피벗 확정 기준 — 룩어헤드 없음)
[자리]   ① 수평 지지/저항  ② 피보 61.8~78.6 되돌림  ③ 볼린저 밴드 밖→재진입(밴드 수평일 때만)
[방아쇠] ⑮ 캔들 반전(장악형/핀바)  ⑯ MACD 크로스/히스토그램 틱룰
[체결]   신호봉 종가 확인 → 다음 봉 시가 진입, ATR 버퍼 손절, R배수 익절, 최대 40봉 보유
```

## 실행

```bash
pip install -r requirements.txt
python gridsearch.py        # 336개 조합 전수 실행 → results/
```

데이터: `data/` 폴더에 CSV를 넣으면 자동 인식 (컬럼: Date/Open/High/Low/Close, 한글·소문자 헤더 허용).
CSV가 없으면 PyPI 번들 실데이터 5종으로 실행: GOOG 일봉 04-13, MSFT 일봉 86-17, S&P500 일봉 99-18, NASDAQ 일봉 99-18, EURUSD 1H 17-18 (약 24,000봉).

## 핵심 발견

1. **자리 필터는 수평 S/R이 압도적.** 유효 조합 평균 PF — S/R 1.05 vs 볼린저 0.59 vs 피보 0.46. 피보/볼린저는 단독으로는 오히려 마이너스였고, TOP 10 전부가 S/R 단독 조합. "아무 자리 다이버전스는 무시하라"는 실전 격언이 수치로 재현됨.
2. **롱만 작동.** 방향별 평균 PF — long 1.34 vs short 0.68. 주가지수/주식의 장기 상승 드리프트 때문에 하락 다이버전스 숏은 이 데이터셋에선 기대값 음수. 숏은 별도 시장(하락장 구간, 원자재)에서 재검증 필요.
3. **트리거는 캔들 > MACD.** 캔들 트리거는 손절이 트리거 캔들 극값이라 리스크가 작고(타이트한 손절 → 같은 R 목표에 유리), MACD는 진입이 늦어 평균 R이 낮았음.
4. **손익비 2.5R이 1.5R보다 우세.** 승률은 40%대로 떨어지지만 기대값은 더 큼 — 변곡을 제대로 잡았을 때 크게 먹는 구조가 맞다는 의미.
5. 1위 조합(`long / S/R / candle / 2.5R / RSI14 / k3`)은 5개 중 4개 인스트루먼트에서 양의 기대값 (S&P500만 음수).

## 주의 (오버피팅 경고)

- 336개를 전수 탐색한 결과이므로 상위권에는 운이 좋았던 조합이 섞여 있음. 실전 적용 전 **아웃오브샘플 검증 필수** — `data/`에 새 CSV(최근 데이터, 다른 시장)를 넣고 상위 조합이 유지되는지 확인할 것.
- 수수료/슬리피지 미반영 (R 기준 성과라 손절 폭이 좁은 캔들 트리거는 체감 영향이 상대적으로 큼).
- 같은 봉에서 손절·익절 모두 도달 시 손절 처리(보수적), 갭 손절은 시가 체결로 -1R보다 나쁜 값 허용.
